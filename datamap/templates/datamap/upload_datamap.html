Getting data into a datamap

Rather than have to type out hundreds of lines of text to create your datamap, you can upload a properly configured CSV file.

Use the form below to select a csv file from your computer to upload into the system.

Required CSV format

You can create a CSV file in MS Excel, Notepad (or any other text editor) or Libreoffice Calc.

It must include the following three headers key, sheet and cell_ref. The rest is up to you.

Don't worry about the Replace all entries checkbox for now - just leave it ticked :-)
